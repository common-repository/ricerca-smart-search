== Description ==
Ricerca is a SMART AND FAST SEARCH that can greatly enhance the user experience for visitors to your website.

It makes finding the content they are looking for faster and more convenient by providing suggestions as they type their search query. This is achieved through the use of algorithms that analyze the website’s content and match it with the user’s input in real-time.

The plugin easy to install and set up. The Mega drop down feature is customizable, allowing you to specify which pages and posts should be included in the search results and how the suggestions are displayed.

The plugin also supports the use of synonyms, which can help expand the search results to include related content that might not be an exact match to the user’s query.

This can result in a more comprehensive search experience for visitors, as they are able to quickly find the information they are looking for, even if their original search query was not an exact match.

Additionally, the plugin has an intuitive user interface that makes it easy to use and customize, making it accessible to both experienced and novice users. Overall, a smart search plugin is a valuable tool for improving the search functionality of your website and providing a better experience for visitors.

A smart search plugin functionality can greatly enhance the search experience for users on a website. This type of plugin provides an intuitive interface that suggests search results as the user types their query, making it easier for them to find what they are looking for. Additionally, a smart search plugin can use advanced algorithms to deliver more relevant results, even if the user’s search query is incomplete or misspelled.

Ricerca has been tested on a site that contains 4000+ products.

= OVER 20+ Advanced features =
Ricerca includes a variety of 20+ features that improves site's conversion and user experience:

- **Customize**. Advanced customized Search Results.
- **Fast**. Super fast search results.
- **High performance**. Not affect on page load. Ricerca tested on a live site with 4000+ products.
- **Reports**. See what visitors intersted on.
- **Popular terms**. Share popular terms between visitors.
- **Search Categories**. Categories and products results.
- **Collection creator**. Create search items sets (e.g: summer collection).
- **Terms corrector**. Monitor and review this feature to find terms which leads to "Not found results" on your site in order to suggest and show alternative results by fixing terms.
- **Promotions system**. Create discover items widgets base on visitors search queries.
- **Items ordering**. Control dynamic items order (e.g: most clicked).
- **Terms blocking**. Block unwanted search terms.
- **Woocommerce and Elementor compatible​**. Full compatible with Woocommerce and Elementor and more themes.
- **Add to cart**. Allow visitors to add products to cart, directly from search results.
- **Auto suggest​**. Suggest terms based on visitors search.
- **Name crossing**. Show product search results from the category whose it's named meets the requested term.
- **Exclude out of stock**. Prevent out of stock product to show in search results.
- **RTL / LTR support**.

== Frequently Asked Questions ==
= How Do I Customize Search Results? =

on the plugin’s settings it is possible to decide which posts and taxonomies to sync. Also under “Posts indexing options” it is possible to chose what to index – content, sku, post’s name, post’s excerpt and etc’

= Does the plugin support rtl? =

Yes.

= Is Ricerca Compatible With Elementor ? =

once your site is using elementor “search form” widget Ricerca knows how to integrate  it so you will be able to use Ricerca within the widget. On the premium plan it soon will be possible to add Ricerca search widget to your elementor site!

= How Do I Block Terms From Appearing Under "Popular Terms"? =

Under “Miscellaneous” it is possible to type terms you would like to block.

= Can I Test Ricerca Before Going Live? =

Sure you can! Just go to “General” and check the “Preview mode” checkbox

= Is Ricerca Compatible With Woocommerce? =

Yes, the plugin has Woocommerce compatibility where it is possible to turn on and off “Add to cart” and “Price” features.

= Is Ricerca Compatible With GDPR? =

Yes, the plugin does not collect any personal data of visitors in any way. and doesn't send data to 3rd party.

= Is Ricerca Compatible With PolyLang plugin? =

Yes, the plugin show results according the current language.

= I have Woocommerce, Sale price remains after sale ends? =

This and Woocommerce issue, please verify woocommerce_scheduled_sales. some times, 
the event deleted entirely from a WordPress Cron jobs.
